API Reference - Friends Search

methode search()

search sentences by specific query, and filtered by language

Parameters

query|q[String, Required]

The text that you want to search. could be any string query, word, sentence and quote. you can avoid special characters like '"!?, and spaces, but we'll do it for you anyway.

language|lang[String, Optional]

You can provide a language to search with, from the supported languages.

limit|l[Int, Optional]

Also, you can tell us how many results do you want to be displayed. Default to 50. more or less can be chosen.

Examples

https://example.com/search?query=unagi&language=en&limit=5

The request will search the word "unagi", in the english language, and will show only the first 5 results.

https://exmple.com/search?q=Hanukkah

Here, you'll get results for the word Hanukkah, in any language, and all of them, unless they will be more than 50.

In the response, you'll get list of results, each one included data about the results, such us:

  • content

    the content of the sentence, just like it showed on TV.

  • start

    The exact second that the sentence appeared in the series.

  • end

    The end of the exact time that the sentence appeared in the series.

  • season

    the number of the season.

  • episode

    the number of the episode.

Errors

Not many errors may appear, but here are the ones:

  • a `query` is required.

    You must provide a query parameter. it could be anything, but have to be here.

  • `language` must be two characters, e.g. `EN`.

    The language parameter must be from the supported languages., otherwise nothing will show. However, if you provide a language parameter that include digits, the error will be raised.

    NOTE: For most of the cases, you don't have to pass the language parameter, but good to know he will be there...
